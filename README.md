# 프로젝트 소개
인공지능 기반 영화 추천 웹사이트 (AI 추천 + 검색 + 순위 확인 기능 포함)

# 기술 스택
백엔드
사용 언어 : Python 
프레임워크 : FastAPI
서버 실행 : Uvicorn
환경 변수 : python-dotenv (.env)
데이터 소스 및 서비스 : TMDB_API, 
KOBIS API, OpenAi API,  
Hugging Face Transformers
프론트엔드
빌드/런타임 : Vite, node.js
프레임워크 및 라이브러리 : React, TypeScript, Tailwind CSS

# 주요 기능
1. AI 감정 기반 영화 추천
2. 영화 순위 조회 (TMDB, KOBIS, IMDB)
3. 영화 검색 기능
